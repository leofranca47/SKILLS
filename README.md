# Skills Repository

Repositório para armazenar links de skills para agentes de IA.

## O que são skills?

Skills são capacidades reutilizáveis de agentes de IA com conhecimento procedimental. Elas permitem que IAs realizem tarefas específicas com mais eficiência e precisão.

## Como usar

Instale uma skill usando:

```bash
npx skillsadd <owner/repo>
```

## Recursos

- Catálogo de skills disponíveis: https://skills.sh/
- Integração com Laravel: https://skills.laravel.cloud/
