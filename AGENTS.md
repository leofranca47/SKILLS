# AGENTS.md

This is a minimal skill-definition repository for the Skills ecosystem.

## What this repo is

- Points to the Skills agent capability registry: https://skills.sh/
- Laravel integration: https://skills.laravel.cloud/

## Working here

- This repo only contains `skill.txt` with skill URLs - no source code
- Install skills with: `npx skillsadd <owner/repo>`
- See https://skills.sh/ for available skills
